Support
=======

Find out how you can get support for Yoast SEO!

Slack
-----
Join the #yoast-seo-for-typo3 channel on the TYPO3 Slack.

If you are not registered yet, `find out more about how to join. <https://typo3.org/community/meet/chat-slack>`_

Github
------
Found a bug in the extension? Please report this on our Github repository:

https://github.com/Yoast/Yoast-SEO-for-TYPO3

Yoast Support
-------------
If you purchased the premium extension, you'll have 24/7 access to the awesome support team at Yoast!

`Find out more on the dedicated page on Yoast.com <https://yoast.com/help/support/#premium>`_
