Contribution
============

Anyone is welcome to contribute to Yoast SEO for TYPO3.

There are various ways you can contribute:

*  `Raise an issue <https://github.com/Yoast/t3ext-yoast-seo/issues>`__ on GitHub.
*  Create a pull request with your bug fixes or new features.
*  Improve the `documentation <https://docs.typo3.org/p/yoast-seo-for-typo3/yoast_seo/main/en-us/>`__
   via the "Edit on GitHub" workflow.

Please `read the guidelines <https://github.com/Yoast/Yoast-SEO-for-TYPO3/blob/master/.github/CONTRIBUTING.md>`__
on how to contribute to this repository and how to report bugs, and take a look
at the ":ref:`development`" page for setting up a local development environment.
